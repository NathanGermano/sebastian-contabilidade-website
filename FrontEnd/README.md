# 💼 Sebastian Contabilidade

![Project Status](https://img.shields.io/badge/Status-Completed-green) ![License](https://img.shields.io/badge/License-MIT-blue) ![HTML5](https://img.shields.io/badge/Tech-HTML5-orange) ![JavaScript](https://img.shields.io/badge/Script-Vanilla%20JS-yellow) ![Sass](https://img.shields.io/badge/Style-SCSS-pink) ![Swiper](https://img.shields.io/badge/Library-Swiper.js-blue)

Projeto de site institucional emulando um escritório de contabilidade, com o objetivo de apresentar a empresa, seus serviços, planos e facilitar o contato com clientes.

# 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação distribuída em múltiplas páginas para organizar o conteúdo institucional.

* **SCSS**: Pré-processador CSS utilizado para uma melhor organização e modulularizção do código.

* **JavaScript (Vanilla):** Utilizado para manipulação da DOM, interatividade de menus e uso dos componentes globais.

* **Swiper.js:** Biblioteca utilizada para a criação de carrosséis modernos e responsivos.

# ⚙️ Funcionalidades

* Navegação Multi-páginas: Estrutura completa com páginas dedicadas para Início, Sobre, Serviços, Planos e Contato.

* Componentização via JS: Script dedicado para adicionar o Cabeçalho e o Rodapé em todas as páginas para melhor organização e manutenção.

* Menu Interativo: Funcionalidade de toggle para navegação mobile e interatividade do usuário com o menu principal.

* Carrosséis Dinâmicos: Integração com Swiper.js para exibir seções com rolagem.

* Design Responsivo: Layout adaptável para desktop e mobile, construído utilizando CSS/SCSS.
